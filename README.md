## 개요
- 프로젝트 이름 : JS로 선택 text 옮기기, checkbox, card 게임 구현하기
- 프로젝트 기간 : 2023.04.17 ~ 2022.04.23
- 개발 언어    : JS(ES6), html, css
- 사용 tool   : vscode

## 기획
- 기획의도 : 실제 웹사이트에서 빈번하게 사용되는 기능을 실제로 구현하고, JS class를 이용한 card 게임 구현

## 실행환경
- Tool : vscode
- Language : html5, css3, JS(ES6)
